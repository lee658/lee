원시 코드 { c, java 같은 프로그램 언어로 작성한 프로그램
컴퓨터가 이해할수 있눈 기계어로 변환해야 실행 가능
기계어로 구성된 프로그램은 원시 코드가 제공 안됨
}

가상화 {
    믈리적인 컴퓨터 자원의 상호 작용, 운영 방식을 사용자에게 감추는 기술
}

플랫폼 {
     앱을 쉽고 편하게 사용하게 지원하는 하드/소프트웨어 환경
}

네트워크 {
    2개 시앙의 컴퓨터가 연결된 구조(가장 작은 네트워크)
}

다형성(Polymor-phism) {
    그 프로그래밍 언어의 자료형 체계의 성질을 나타내는 것으로, 프로그램 언어의 각 요소들이
    다양한 자료형(type)에 속하는 것이 허가되는 성질
}
