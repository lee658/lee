    시스템 소프트웨어
시스템 정의 : 하나의 공통적인 목적울 가지고 있는 구성 요소 들을 상호 결합해 놓은
유기적인 집합체 이다

    시스템의 기본 요소 
입력(Input) : 처리 방법, 제어 조건, 처리할 데이터 투입 요소
출력(Output) : 처리된 결과를 시스템에 출력하는 요소
처리(Process) : 입력된 자료를 처리 조건에 따라 변환, 가공 하는 요소
제어(Control) : 시스템의 기본 요소들이 각 행동과정 감시 요소 
피드백(Feedback) : 목표의 불충분할 경우 다시 입력하는 요소  {
    전수 : 출력되는 신호를 전부 피드백
    부분 : 지정된 일부의 내용만 제한 시간 내에 피드백
  전력적 : 핵심적인 단위 내용 선택 지정대로 피드백
    기호 : 기호가 붙은 사항만 피드백
}

CPU관리, 메모리관리, 디스크 파일 관리하는 소프트웨어
하드웨어의 구조, 특성을 몰라도 컴퓨터를 이용할 수 있도록 지원
