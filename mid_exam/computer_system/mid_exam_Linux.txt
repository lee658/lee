1. 컴퓨터 호스트 게스트
호스트 : 실제 컴퓨터 > 이미 컴퓨터에 설치된 os
게스트 : 가상 머신(가상 컴퓨터) > 가상 머신에 설치할 그어ㅚ 운영체제

2. 유닉스 운영체제 특징
c언어 작성되어 이식성과 장치 프로세스간의 호환성 높다
소스공개괸 개방형 시스템
크기가 작아 이해하기가 쉽다
다중 사용자, 다중 작업 지원
통신망관리용 운영체제로 좋다
트리구조 파일 시스템

3. cpu 코어
기억, 해석, 연산, 제어 역활을 하는 컴퓨터 중앙 처리 장치
(멀티코어를 늘려 성능 증폭(듀얼 쿼드 헥사...))

4. 쉘
사용자의 입력을 커널로 보낸후 커널에서 온 결과를 사용자에게 출력

5. 스냅숏
가상머신에서 중요한 파일을 삭제했을때 복구하는 기능

6. 멀티부팅 / 가상머신 
멀티부팅 : 하드디스크나 SSD의 파티션을 분할하고 한 번에 하나의 
운영체제만 가동할 수 있는 환경
가상머신 : 파티션을 나누지 않고 여러개 운영체제 가동
*멀티부팅 파티션 분할 / 가상머신 파티션 나누지 않음

7. GNU 프로젝트 목적
모두가 공유 할 수 있는 소프트웨어를 만드는것

8. GPL 라이선스
소프트웨어 사용에 대한 자유
소프트웨어 수정에 대한 자유
소프트웨어 재배포에 대한 자유
수정된 소프트웨어 이익을 전체가 얻을 수 있도록 배포 할 수 있는 자유

9. 런레벨 
0 종료
1 시스템 복구 모드 단일 사용자
2 (멀티유저)        사용안함 
3 텍스트, 다중 사용자 모드 
4 (멀티유저)    사용안함
5 그래픽모드의 다중 사용자 모드
6 reboot

10. 커널  
메모리 관리: 메모리가 어디서 얼마 사용되는지 추적
프로세스 관리: 중앙 처리 장치(CPU) 사용시간 결정
장치 드라이버: 하드웨어 프로세스 중재자 역할
시스템 호출, 보안: 프로세스의 서비스 요청 수신

11. 쉘(cmd, termal) 명령어
ipconfig /all : ip주소,서브넷마스크 등 보여주는 명령어
cd : 현재 디렉토리에서 이동하는 명령어
ls: 현재 디렉토리의 있는 파일/디렉토리 를 보여줌
clear: 터미널창 텍스트 삭제
sudo su -root : root 계정 활성화(sudo : 루트 권한으로 실행)
종료 명령어 : shutdown -p now, poweroff, init 0, halt -p


무료 유닉스 = 리눅스
우분투 종류 : 쿠분투, 우분투 그놈, 우분투 기린, 루분투
