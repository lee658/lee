    DBSM(DataBase Management System)
 
    DBSM의 정의
-대량의 데이터를 저장하고 있는 데이터베이스를 생성, 조회. 변경 등 관리/변경 하는 프로그램
-유저, 앱, 데이터베이스와 상호 작용하여 데이터를 저장하고 분석하기 위한 앱

    DBSM의 종류
Oracle, IBm DB2 : 대규모 데이터 처리, 안정적인 처리
SQL Sever : 중소 규모 데이터 처리, 안정적 처리
MySQL : 오픈소스에서 주로 사용되는 RDBMS
SQLite : 스마트폰, 태블릿 PC 등의Embedded 데이터베이스 용도로 사용
MongoDB : 오픈소스이며 NoSQL DBMS 이다
Redis : 오픈소스 키-값(key-value)DBSM 이다

    DBSM 분석시 고려사항
가용성 : 오랜 기간 시스템을 운영할 때 장애 발생 가능성이 있는지 확인

성능 : 대규모 데이터를 처리할 만한 성능 확인
다양한 튜닝 옵션 지원 확인

기술 지원 : 공급 벤더들의 안적인 기술을 지원 확인
오픈소스 여부 확인

상호 호환성 : 설치 가능 운영체제 확인
다양한 운영체제에서 지원되는 JDBC, ODBC 확인

구축 비용 : 라이선스 정책, 비용 확인 유지/관리 비용 확인
총 소요 비용(TCO)확인
